## Vector Viewer App
Created a first-draft standalone HTML app at `vector-viewer.html` for OpenClaw-driven evolving visual output.
- Displays a square Cartesian frame with grid and axes.
- Starts with one vector at `(0.0, 0.0, 0.0)` with `L=1.0`.
- Exposes a simple `window.VectorViewer` API for future updates:
  - `getState()`
  - `setVectors(vectors)`
  - `addVector(vector)`

This is intended to run alongside OpenClaw and be updated over time.
